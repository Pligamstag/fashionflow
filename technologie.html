6. Technologie et plan de développement

Langages de programmation

  • JavaScript : pour la logique globale de l'application.
  • React Native : pour développer une application mobile multiplateforme (Android et iOS).

Base de données

  • Firebase : pour l’authentification, le stockage des données utilisateur (profils, préférences, historique), et la gestion en temps réel des interactions.

Backend (si nécessaire)

  • Node.js avec Express : pour la gestion des requêtes API et la logique serveur.

Outils supplémentaires

  • Expo : pour simplifier le développement et le déploiement.
  • Git/GitHub : pour la gestion du code source et la collaboration.

Plateformes visées

  • Android
  • iOS

Grâce à React Native, une seule base de code sera utilisée pour cibler les deux systèmes d'exploitation mobiles.
